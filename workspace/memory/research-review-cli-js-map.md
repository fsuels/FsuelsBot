## Top 5 Recommended Actions

1. **Implement a “source-reachability gate” in research workflows (P0).**  
   Before synthesis, auto-check URL DNS/HTTP reachability and mark each source as retrievable/unretrievable. This prevents fabricated interpretation from dead links.

2. **Adopt the local `memory-consolidation` integrity pattern as baseline policy (P0).**  
   Keep checkpoint-first incremental extraction, binding-event non-decay, and mandatory integrity report (`memory/integrity-report.json`) as hard guardrails.

3. **Add a URL-to-local-mirror fallback map for known internal docs (P1).**  
   If URL fails but an explicitly mapped local canonical file exists, analyze local mirror with a visible “mirror used” flag; otherwise default IGNORE.

4. **Run a limited experiment syncing high-credibility official docs (`claude.md` redirect target) into internal guidance (P1).**  
   Weekly pull + human-reviewed diff; no automatic policy replacement. Treat as advisory until 2-4 week stability is observed.

5. **Standardize decision scoring semantics in research tasks (P2).**  
   Keep 0-5 scale definitions explicit (benefit vs burden axes) to improve consistency across revenue/autonomy/reliability/cost/risk decisions.
