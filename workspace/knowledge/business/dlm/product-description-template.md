# DressLikeMommy — Product Description Template (PDP)

Purpose: a copy/paste-ready structure for Shopify product descriptions that is:

- conversion-first (clear benefits + reduced anxiety)
- SEO-safe (keyword presence without stuffing)
- operations-safe (no unverifiable claims)

## Epistemic guardrails (MANDATORY)

- **No made-up specifics**: if a detail isn’t in supplier/Shopify data, write it as optional placeholder or omit.
- **No absolute claims** unless we can prove them ("best", "#1", "guaranteed delivery by X" etc.).
- **Shipping & returns**: reference your actual policy pages; don’t invent timelines.

---

## Template (paste into Shopify Description)

### 1) One-line outcome (hero line)

**Make every day feel put-together—without trying too hard.**

> Replace with the real primary benefit/outcome.

### 2) Quick promise (2–3 bullets)

- **Flattering, easy-to-wear fit** that works for real life
- **Soft feel** for all-day comfort
- **Style-ready** for school pickup, errands, brunch, and weekends

> Only keep bullets you can support from actual material/fit info.

### 3) Why you’ll love it (short paragraph)

This piece is designed for effortless outfits: throw it on, feel confident, and go. Pair it with sneakers for casual days or dress it up with boots and accessories.

### 4) Key details (scannable list)

- **Style:** [e.g., Midi dress / Lounge set / Sweater]
- **Neckline:** [ ]
- **Sleeves:** [ ]
- **Length:** [ ]
- **Fabric:** [material % if known]
- **Stretch:** [none / slight / medium / high]
- **Care:** [machine wash cold / hang dry] (only if known)

### 5) Fit notes (reduce sizing anxiety)

- **Fit:** [true to size / relaxed / fitted]
- **Model info:** [height + size worn] (only if verified)
- **If between sizes:** consider sizing [up/down] (only if consistent with supplier guidance)

### 6) How to wear (2 outfit ideas)

- **Casual:** [top] + [bottom] + sneakers
- **Elevated:** [layer] + [shoe] + [bag]

### 7) What you get (expectation-setting)

- Includes: **1× [product name]**
- Color: **[color name]** (selectable)
- Sizes: **[size range]**

### 8) Shipping & returns (link, don’t invent)

For shipping times and returns, see:

- **Shipping Policy:** /policies/shipping-policy
- **Refund Policy:** /policies/refund-policy

### 9) SEO footer (1 natural sentence)

Keywords: [primary keyword], [secondary keyword], [occasion/season keyword].

> Keep this to **one** natural sentence; do not keyword-stuff.

---

## Mini-checklist (before publishing)

- [ ] Title matches the actual item (no "100% cotton" unless verified)
- [ ] At least **1** clear benefit and **1** scannable details section
- [ ] Fit + sizing guidance present (or explicitly omitted due to unknown)
- [ ] Policy links included
- [ ] No unverifiable trust claims ("verified", "best seller" unless real)
