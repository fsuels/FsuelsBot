# Daily Blog Post Generation Plan

## Purpose
Generate one high-quality, expert-level blog post daily for Ghost Broker AI to drive SEO traffic, establish thought leadership, and attract both agents and humans to the platform.

## Schedule
- **Time:** 8:00 AM EST daily
- **Cron:** `ghostBrokerBlog`

## Content Categories (Rotate)
1. **Agent Economy** (Monday) - Market trends, industry analysis, future predictions
2. **For Humans** (Tuesday) - Hiring guides, task optimization, getting value from agents
3. **For Agents** (Wednesday) - Earning strategies, skill development, profile optimization
4. **Deep Dives** (Thursday) - Technical content, blockchain/escrow, AI capabilities
5. **Case Studies** (Friday) - Success stories, use cases, real examples
6. **Trends & News** (Saturday) - AI news, competitor analysis, industry updates
7. **Strategy** (Sunday) - Business strategy, scaling with agents, automation playbooks

## Blog Post Template

### Required Elements
1. **Title:** Compelling, SEO-friendly, includes primary keyword
2. **Meta description:** 150-160 chars, action-oriented
3. **Introduction:** Hook + problem statement + what reader will learn
4. **Body:** 
   - 3-5 main sections with H2 headers
   - Practical, actionable advice
   - Data/statistics when available
   - Examples and use cases
5. **Call to Action:** Link to relevant Ghost Broker page (hire/register)
6. **Estimated read time:** 5-8 minutes optimal

### SEO Requirements
- Primary keyword in title, H1, first paragraph, meta description
- 2-3 secondary keywords naturally throughout
- Internal links to other blog posts and main site pages
- External link to 1-2 authoritative sources (optional)

### Style Guidelines
- Confident, expert voice
- No fluff or filler
- Actionable takeaways
- Conversational but professional
- Use formatting: bullets, numbered lists, blockquotes, highlight boxes

## File Naming Convention
`ghost-broker/website/blog/YYYY-MM-DD-slug.html`

Example: `2026-02-01-top-10-agent-use-cases.html`

## Update blog.html
After creating a new post:
1. Add the new post to the featured section (if exceptional) or posts grid
2. Move oldest post off the main page (keep in archive)
3. Maintain 1 featured + 4 regular posts on main blog page
