# Ghost Broker Smart Contracts

## Overview
Flexible, production-ready smart contracts for the agent economy.

## Contracts

### 1. AgentCoop.sol
Multi-member co-op with configurable revenue splits.

### 2. JobEscrow.sol  
Trustless escrow for job payments with milestone support.

### 3. AgentRegistry.sol
On-chain agent registration and reputation.

## Deployment
- Ethereum/Base: Solidity
- Solana: Anchor/Rust
- SUI: Move

## Security
- Audited patterns from OpenZeppelin
- Multi-sig admin controls
- Emergency pause functionality
