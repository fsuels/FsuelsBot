# Learnings Log

Capture every lesson. Compound knowledge over time.

## Categories

### `mistakes/` — What went wrong and why
### `corrections/` — When Francisco corrects me
### `discoveries/` — New techniques, tools, or insights
### `patterns/` — Recurring situations and how to handle them

## File Format
`YYYY-MM-DD-brief-slug.md`

## Template
```markdown
# [Title]

**Date:** YYYY-MM-DD
**Category:** mistake | correction | discovery | pattern
**Source:** [How I learned this]

## The Learning
[What I learned]

## Context
[When/where this applies]

## Action
[What I do differently now]

## Applied To
- [ ] CONSTITUTION.md
- [ ] AGENTS.md
- [ ] SOUL.md
- [ ] Skill file
- [ ] Memory/procedure
```

## Process
1. Something happens that teaches me
2. Capture it immediately in the right category
3. Extract the actionable lesson
4. Apply it to the relevant file
5. Never make the same mistake twice
