# Product Listing SEO Procedure

> **Verification Gate:** Before proceeding, state: "I have read the product listing SEO procedure. Every product needs: optimized title, keyword-rich description, alt-tagged images, and proper categorization."

## Purpose
Optimize product pages for search visibility and conversions.

## Prerequisites
- Keyword map completed (from `keywords.md`)
- Product imported to Shopify
- Product images ready

## Tools/Resources
- Shopify Admin > Products
- Keyword map document
- Competitor product pages for reference

---

## Step-by-Step Procedure

### Step 1: Optimize Product Title
- [ ] **Format:** `[Key Feature] [Product Type] - [Target Audience] [Optional: Style]`
  - Example: `Heart Print Matching Sweatshirts - Mommy and Me Set`
- [ ] Include primary keyword naturally
- [ ] Keep under 70 characters for full display
- [ ] Avoid ALL CAPS or excessive punctuation
- [ ] Don't include price or brand (those show separately)

### Step 2: Write SEO-Optimized Description
Structure your description:

```markdown
## Opening Hook (Include primary keyword)
[1-2 sentences describing the product and its appeal]

## Features & Benefits
- Feature 1: [Benefit]
- Feature 2: [Benefit]
- Feature 3: [Benefit]

## Sizing Information
[Size chart or fit guidance]

## Care Instructions
[Washing/care details]

## Perfect For
[Occasions, uses, gift ideas]
```

**Example:**

### Step 3: Optimize Product URL Handle
- [ ] Go to Search engine listing preview > Edit website SEO
- [ ] Set URL handle to include keywords
  - Good: `heart-print-mommy-me-matching-sweatshirt`
  - Bad: `product-12345` or `red-top`
- [ ] Use hyphens between words
- [ ] Keep concise but descriptive

### Step 4: Add Product Tags
Tags help with internal search and organization:
- [ ] Add category tag: `matching-sets`, `sweatshirts`, `tops`
- [ ] Add occasion tag: `valentines`, `everyday`, `holiday`
- [ ] Add style tag: `hearts`, `pink`, `red`
- [ ] Add size tag: `mommy-sizes`, `kids-sizes`

### Step 5: Set Product Category (for Google Shopping)
- [ ] In Product > Organization > Product category
- [ ] Select most specific category:
  - Apparel & Accessories > Clothing > Shirts & Tops
  - Or: Apparel & Accessories > Clothing > Sleepwear & Loungewear
- [ ] This feeds into Google Merchant Center

### Step 6: Add to Appropriate Collections
- [ ] Add to main collection: All Products
- [ ] Add to category collection: Sweatshirts, Tops
- [ ] Add to occasion collection: Valentine's Day
- [ ] Add to featured collection if applicable: New Arrivals, Best Sellers

### Step 7: Complete Image SEO (See images.md)
- [ ] All images have descriptive filenames
- [ ] All images have alt text with keywords
- [ ] First image is the best (it shows in search results)

### Step 8: Add Product-Level Schema (Automatic in Shopify)
Shopify automatically adds Product schema, but verify:
- [ ] Price is visible
- [ ] Availability status is correct
- [ ] Product has reviews (if applicable)

## Description Keyword Placement

**Must include primary keyword in:**
- [ ] First 100 characters of description
- [ ] At least one subheading
- [ ] Naturally 2-3 times throughout (don't stuff)

**Secondary keywords in:**
- [ ] Feature bullets
- [ ] "Perfect for" section
- [ ] Closing paragraph

## Shopify Navigation Path
1. Products > Add product (or select existing)
2. Title field at top
3. Description in rich text editor
4. Media section for images
5. Organization section: Product category, Type, Vendor, Collections, Tags
6. Search engine listing preview: Click "Edit" for URL, meta title, meta description
