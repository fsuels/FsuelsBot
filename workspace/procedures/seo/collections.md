# Collection Page SEO Procedure

> **Verification Gate:** Before proceeding, state: "I have read the collection SEO procedure. Each collection needs: keyword-optimized title, 100+ word description, and proper meta tags."

## Purpose
Optimize collection pages to rank for category-level keywords.

## Prerequisites
- Keyword map completed (from `keywords.md`)
- Collections created in Shopify
- Products added to collections

## Tools/Resources
- Shopify Admin > Products > Collections
- Keyword map document

---

## Step-by-Step Procedure

### Step 1: Plan Collection Structure
Organize collections by:
- [ ] **Product Type:** Dresses, Sweatshirts, Pajamas, Swimwear
- [ ] **Occasion:** Valentine's Day, Easter, Christmas, Everyday
- [ ] **Style:** Matching Sets, Coordinating Outfits
- [ ] **Audience:** Mommy & Me, Father & Son, Family Sets

### Step 2: Optimize Collection Title
- [ ] Include primary keyword naturally
- [ ] Keep descriptive but concise
- [ ] **Format:** `[Primary Keyword] [Modifier]`

**Examples:**
| Good Title | Primary Keyword |
|------------|-----------------|
| Valentine's Day Mommy and Me Outfits | valentine mommy and me outfits |
| Matching Mother Daughter Dresses | matching mother daughter dresses |
| Mommy and Me Pajamas | mommy and me pajamas |
| Christmas Family Matching Sets | christmas family matching |

### Step 3: Write Collection Description
- [ ] Minimum 100 words (150-200 recommended)
- [ ] Include primary keyword in first sentence
- [ ] Include 2-3 secondary keywords naturally
- [ ] Add value proposition and shopping guidance
- [ ] Use formatting (headers, bullets) for readability

**Template:**
```markdown
# [Collection Name]

[Opening paragraph with primary keyword - what this collection is, who it's for]

## What You'll Find

[Description of products in collection, variety, styles available]

## Why Shop [Collection Name] at Dress Like Mommy

- [Benefit 1]
- [Benefit 2]
- [Benefit 3]

## Perfect For

[List occasions, uses, gift ideas]

[Closing sentence with CTA and keyword]
```

**Example - Valentine's Collection:**
```markdown
Celebrate love and togetherness with our adorable **Valentine's Day Mommy and Me Outfits**! From cozy matching sweatshirts to elegant red dresses, find the perfect Valentine's look for you and your mini.

## What You'll Find

Our Valentine's collection features heart prints, pink and red palettes, and love-themed designs perfect for:
- Valentine's Day photo shoots
- Mommy-daughter date nights
- Family celebrations
- School Valentine's parties

## Why Shop Valentine's Outfits at Dress Like Mommy

- ❤️ Curated matching sets sized for moms and kids
- ❤️ Quality fabrics that photograph beautifully
- ❤️ Free shipping on orders over $50
- ❤️ Easy returns if sizing isn't perfect

Make this Valentine's Day extra special with matching outfits that show off your bond. **Shop our Valentine's mommy and me collection now!**
```

### Step 4: Optimize Meta Tags
- [ ] Go to Collection > Edit > Search engine listing preview
- [ ] Set **Page title** (50-60 chars):
  - `Valentine Mommy and Me Outfits | Dress Like Mommy`
- [ ] Set **Meta description** (150-160 chars):
  - `Shop Valentine's Day mommy and me outfits - matching dresses, sweatshirts & sets. Perfect for photos! Free shipping on $50+. ❤️`
- [ ] Set **URL handle**:
  - `valentines-mommy-and-me` (not `valentines-day-collection-2025`)

### Step 5: Set Collection Image
- [ ] Upload a high-quality collection banner image
- [ ] Image should showcase collection theme
- [ ] Add alt text: `Valentine's Day mommy and me matching outfits collection`

### Step 6: Configure Collection Products
- [ ] Ensure all relevant products are in collection
- [ ] Set **sort order**: Best Selling, Newest, or Manual
- [ ] If manual: Put best products first (above the fold)

### Step 7: Add Internal Links
In the collection description, link to:
- [ ] Related collections
- [ ] Best-selling products
- [ ] Blog posts about the topic

## Collection Description Formulas

**Occasion Collection:**
```
Celebrate [occasion] with our [keyword collection name]! From [product type] to [product type], find matching looks perfect for [use cases]. Shop [occasion] matching outfits now.
```

**Product Type Collection:**
```
Discover our collection of [keyword - product type]. We've curated the best [adjective] [products] for mommy and me, featuring [styles/features]. Perfect for [occasions].
```

**Seasonal Collection:**
```
[Season] is here! Browse our [keyword seasonal collection] featuring [product types and styles]. These matching [season] looks are perfect for [activities/occasions].
```

---

## Quality Criteria
✅ Collection title includes primary keyword  
✅ Description is 100+ words with natural keyword usage  
✅ Meta title is 50-60 characters with keyword  
✅ Meta description is 150-160 characters with CTA  
✅ URL handle is clean and keyword-rich  
✅ Collection has banner image with alt text  
✅ Products are properly sorted  

---

## Common Mistakes to Avoid
❌ Empty or missing collection descriptions  
❌ Duplicate descriptions across collections  
❌ Generic titles like "New Collection" or "Sale"  
❌ URL handles with dates that expire (`valentines-2025`)  
❌ Missing collection images  
❌ Not updating meta tags (defaults to title only)

## Shopify Navigation Path
1. Products > Collections > Create collection (or select existing)
2. Title field at top
3. Description in rich text editor
4. Collection image on right side
5. Collection type: Manual or Automated
6. Search engine listing preview: Click "Edit" at bottom
